4.1 Sayfa Render Akışı
┌─────────────────────────────────────────────────────────────────────────────┐
│                         SAYFA RENDER AKIŞI                                   │
└─────────────────────────────────────────────────────────────────────────────┘

    HTTP Request (GET /hizmetlerimiz)
           │
           ▼
    ┌──────────────────┐
    │  SetLocale       │──────► Session/Cookie/URL'den dil belirleme
    │  Middleware      │
    └────────┬─────────┘
             │
             ▼
    ┌──────────────────┐
    │  Route           │──────► Route::get('/{slug}', [PageController::class, 'show'])
    │  Resolution      │
    └────────┬─────────┘
             │
             ▼
    ┌──────────────────┐
    │  PageController  │
    │  show()          │
    └────────┬─────────┘
             │
             ▼
    ┌──────────────────┐       ┌──────────────────┐
    │  PageService     │◄──────│  Cache Layer     │
    │  getBySlug()     │       │  (Redis/File)    │
    └────────┬─────────┘       └──────────────────┘
             │
             │ Cache Miss
             ▼
    ┌──────────────────┐
    │  PageRepository  │
    │  findBySlug()    │
    └────────┬─────────┘
             │
             ▼
    ┌──────────────────┐
    │  Page Model      │──────► with(['sections' => orderBy('order')])
    │  + Sections      │
    └────────┬─────────┘
             │
             ▼
    ┌──────────────────┐
    │  SectionRenderer │
    │  renderAll()     │
    └────────┬─────────┘
             │
             │ Her section için:
             ▼
    ┌──────────────────────────────────────────────────────┐
    │                                                       │
    │  ┌──────────────────┐                                │
    │  │ SectionRegistry  │──────► Section config al        │
    │  └────────┬─────────┘                                │
    │           │                                          │
    │           ▼                                          │
    │  ┌──────────────────┐                                │
    │  │ DataHandler?     │──────► Varsa dinamik veri çek   │
    │  │ (Optional)       │        (Posts, Products, etc.)  │
    │  └────────┬─────────┘                                │
    │           │                                          │
    │           ▼                                          │
    │  ┌──────────────────┐                                │
    │  │ TranslationServ. │──────► JSON'dan mevcut dil      │
    │  │ resolveContent() │        değerlerini çöz          │
    │  └────────┬─────────┘                                │
    │           │                                          │
    │           ▼                                          │
    │  ┌──────────────────┐                                │
    │  │ Blade View       │──────► view('sections._hero')   │
    │  │ Render           │        ->with($data)            │
    │  └────────┬─────────┘                                │
    │           │                                          │
    │           ▼                                          │
    │       HTML Output                                    │
    │                                                       │
    └──────────────────────────────────────────────────────┘
             │
             ▼
    ┌──────────────────┐
    │  Layout Resolver │──────► Header + Content + Footer
    └────────┬─────────┘
             │
             ▼
    ┌──────────────────┐
    │  View Composers  │──────► Menus, Settings, etc.
    └────────┬─────────┘
             │
             ▼
       Final HTML Response

    HTTP POST /admin/media/upload
    (multipart/form-data)
           │
           ▼
    ┌──────────────────┐
    │  UploadMedia     │──────► Validation: mime, size, dimensions
    │  Request         │
    └────────┬─────────┘

4. Kurulum Adımları
┌─────────────────────────────────────────────────────────────────────────────┐
│                         KURULUM ADIMLARI                                     │
└─────────────────────────────────────────────────────────────────────────────┘

ADIM 1: Laravel Projesi Oluşturma
═══════════════════════════════════════════════════════════════════════════════

$ composer create-project laravel/laravel cms-project
$ cd cms-project

ADIM 2: Temel Paketlerin Kurulumu
═══════════════════════════════════════════════════════════════════════════════

# Filament Admin Panel
$ composer require filament/filament:"^3.2"
$ php artisan filament:install --panels

# Spatie Paketleri
$ composer require spatie/laravel-permission          # Rol & İzin
$ composer require spatie/laravel-activitylog        # Activity Log
$ composer require spatie/laravel-backup             # Backup
$ composer require spatie/laravel-medialibrary       # Media (opsiyonel)
$ composer require spatie/laravel-translatable       # JSON Translations
$ composer require spatie/laravel-sluggable          # Auto Slug
$ composer require spatie/laravel-sitemap            # Sitemap

# Image Processing
$ composer require intervention/image-laravel        # Image manipulation

# SEO
$ composer require artesaos/seotools                 # SEO Tools (opsiyonel)

# Form & Validation
$ composer require biscolab/laravel-recaptcha        # reCAPTCHA

# Utilities
$ composer require league/flysystem-aws-s3-v3        # S3 Storage
$ composer require league/flysystem-ftp              # FTP Storage (opsiyonel)

# Development
$ composer require --dev laravel/telescope           # Debugging
$ composer require --dev barryvdh/laravel-debugbar   # Debug Bar
$ composer require --dev laravel/pint                # Code Style

ADIM 3: Paket Konfigürasyonları
═══════════════════════════════════════════════════════════════════════════════

# Publish configs
$ php artisan vendor:publish --provider="Spatie\Permission\PermissionServiceProvider"
$ php artisan vendor:publish --provider="Spatie\Activitylog\ActivitylogServiceProvider"
$ php artisan vendor:publish --provider="Spatie\Backup\BackupServiceProvider"
$ php artisan vendor:publish --provider="Intervention\Image\Laravel\ServiceProvider"
